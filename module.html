S1 System on Module.

Bluetooth, FPGA, and Battery Management, all in one. Perfect for deploying hardware accelerated algorithms onto tiny form factor IoT devices.

Bluetooth 5.2

Based on Nordic's nRF52, the S1 features Bluetooth 5.2 with extensibility for direction finding, and thread. With on board antenna, you don't need to worry about designing RF hardware into your product. Simply connect power, and the module will be up and running.

The S1 Module is designed for ultimate efficiency. The nRF52 runs your core application, and you will have access to a host of low power modes which can be switched on the fly.

nRF52 Features

Battery Management

To reduce component count in your design. The S1 Module features complete power and battery management. Charge any single cell lithium with the built in battery charger, and make use of four user configurable voltage rails which are broken out. One of which is a buck-boost up to 5.5V.

The power management IC is highly user configurable with software control, so you can dial in your power requirements, or shut down rails when not needed to save energy.

FPGA + DSP

Deploy hardware acceleration wireless with the S1 Module. Save on computation demand that would otherwise need to be done in the cloud. With the FPGA you can deploy realtime signal processing for sensor fusion applications, or run machine learning models.

The iCE40 features 5k LUTs with 8 hardware DSP blocks. There's also 1Mb of SRAM, PLL and two built in oscillators, so you can deploy applications without any external components.

32Mb of user flash serve has boot ROM for the FPGA, as well as user storage. Perfect for in-field updates which can be done wirelessly to enable new FPGA applications on the fly.

An extensive open source toolchain is available for the iCE40, making it easier than ever to get started with FPGA.

IO Connectivity

Easily connect sensors and peripherals for your IoT application. 8 high speed digital IO allow for fast data processing, as well several power rails for flexible power configurations.

‣ 8 digital IO (supporting I3C / I2C / SPI)
‣ 2 analog inputs
‣ 5V Charging input
‣ Lithium cell input
‣ 0.8 - 1.8V LDO output
‣ 0.8 - 3.9V LDO output
‣ 0.8 - 5.5V Buck-Boost output

Applications

The S1 is targeted for Wearables and IoT devices with tight form factors. The 7x12mm footprint makes it perfect for where space is a concern.

Save months of development and avoid HDI PCBs or RF tuning in your design. It's already done for you within the S1. It is a pick-and-place compatible module, suitable for standard FR4 or FPC based designs.
